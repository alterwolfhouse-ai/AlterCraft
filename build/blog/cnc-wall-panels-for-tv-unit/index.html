Before choosing a pattern

  • Check TV size and viewing distance.
  • Leave access for wires, router, set-top box and speakers.
  • Choose panel depth based on room size.
  • Use lighting only where it can be serviced later.

Large patterns suit bigger walls. Smaller repeated textures work better behind TV units where the screen is already the main focus.
